---
title: rv tree
description: List of package dependencies in a tree format
---

Returns the dependency tree of all packages in the project. 

## Usage
```shell
rv tree [OPTIONS]
```

### Options
* `--depth` - Dependency depth to display: 1 - only display root dependencies, 2 - root dependencies + their direct dependencies.
Defaults to displaying everything
* `--hide-system-deps` - Whether to display the system dependencies on each leaf. 
System dependencies are only available on some Linux platforms, otherwise is already hidden.
* `--r-version` - Specify an R version different than the one in the config. The R version does not need to be on the system

## Example
**Config File:**
```toml
// rproject.toml
[project]
name = "tree"
r_version = "4.4"

repositories = [
    { alias = "CRAN", url = "https://cran.r-project.org" }
]

dependencies = [
    "stringr",
    "purrr",
]
```
