Returns the cache root and shows the sub-directory that corresponds to each repository.

### Usage
```shell
rv cache [OPTIONS]
```

### Example
```shell {1}
% rv cache
/Users/rv-user/.cache/rv
PPM (https://packagemanager.posit.co/cran/latest -> f4780dda90), path: /Users/rv-user/.cache/rv/f4780dda90
```
